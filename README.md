# MOPS PDF Packager

台股公開資訊報告一鍵下載工具。輸入股票代碼，自動從公開資訊觀測站 (MOPS) 及 ESG 數位平台下載報告，方便上傳至 Google NotebookLM 進行 AI 分析。

## 下載內容

| 報告類型 | 來源 | 數量 |
|---------|------|------|
| 年報 | doc.twse.com.tw | 最近 5 年 |
| 財報 | doc.twse.com.tw | 最近年度全季 + 歷史年度 Q4 |
| 關係企業三書表 | doc.twse.com.tw | 最近 5 年 |
| 法說會簡報 | mopsov.twse.com.tw | 最近 5 場 |
| ESG 永續報告書 | esggenplus.twse.com.tw | 最近 3 年 |

## 使用方式

### 方式一：批次下載多家公司（推薦）

一次下載多家公司的報告，可自由選擇要下載哪些報告類型：

1. 打開 `公司清單.xlsx`
2. 填入股票代碼、年份區間、勾選要下載的報告類型
3. 雙擊 `批次下載.bat`

**Excel 格式說明：**

| 股票代碼 | 起始年份 | 結束年份 | 全選 | 年報 | 財報 | 關係企業三書表 | 法說會簡報 | ESG永續報告書 |
|---------|---------|---------|------|------|------|-------------|----------|-------------|
| 2330 | | | ✓ | | | | | |
| 2317 | 110 | 115 | | ✓ | | | ✓ | |
| 3008 | | | | | | | | ✓ |

- **年份欄位**（B、C 欄）：民國年，選填。都空白預設抓最近 5 年
- **全選**（D 欄）：打 ✓ 表示下載全部五種報告
- **個別選擇**（E~I 欄）：只在 D 欄留空時有效，打 ✓ 選擇要下載的類型
- **防呆**：如果 D~I 全部留空，自動當作全選處理

**輸出資料夾結構：**

```
MOPS_批次下載/
├── batch_download_YYYYMMDD_HHMMSS.log
├── 2330_台積電/
│   ├── 年報/
│   ├── 財報/
│   ├── 關係企業三書表/
│   ├── 法說會簡報/
│   └── ESG永續報告書/
├── 2317_鴻海/
│   └── ...
└── ...
```

**功能特色：**
- 斷點續傳：中途中斷後重跑，會自動跳過已完成的公司
- 自動限速：每家公司之間等待 45~90 秒，每個年度之間等待 8~15 秒，避免被 MOPS 封鎖
- 自動重試：遇到「查詢過量」封鎖時，自動等待 60 秒後重試（最多 3 次）
- 完整 log：所有下載紀錄同時輸出到 console 和 log 檔案

### 方式二：用 Python CLI（單一公司）

```bash
# 安裝依賴
pip install -r requirements.txt

# CLI 模式
python main.py 2330
python main.py 2330 --year 113
```

## 專案結構

```
mops_pdf_packager/
├── main.py                # CLI 入口（單一公司）
├── batch_download.py      # 批次下載入口
├── 公司清單.xlsx           # 批次下載用的公司清單 template
├── requirements.txt
├── build_exe.bat          # 一鍵打包 EXE
├── 批次下載.bat             # 批次下載 BAT 快捷方式
└── src/
    ├── scrapers/
    │   ├── ebook_scraper.py       # 年報/財報/三書表
    │   ├── briefing_scraper.py    # 法說會簡報
    │   ├── esg_scraper.py         # ESG 永續報告書
    │   └── mopsov_scraper.py      # Selenium 備援
    └── utils/
        └── downloader.py          # 主流程控制
```

## 注意事項

- 部分公司可能沒有 ESG 永續報告書，工具會自動跳過
- 下載速度取決於 MOPS / TWSE 伺服器回應速度，請耐心等待
- 如遇下載失敗，可能是伺服器暫時繁忙，稍後重試即可
- MOPS 伺服器有查詢頻率限制，工具已內建自動限速和重試機制
