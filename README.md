# MOPS PDF Packager

台股公開資訊報告批次下載工具。從公開資訊觀測站 (MOPS) 及 ESG 數位平台自動下載報告，支援自動上傳 GitHub 及同步 Google Drive。

## 下載內容

| 報告類型 | 來源 | 數量 |
|---------|------|------|
| 年報 | doc.twse.com.tw | 最近 5 年 |
| 財報 | doc.twse.com.tw | 最近年度全季 + 歷史年度 Q4 |
| 關係企業三書表 | doc.twse.com.tw | 最近 5 年 |
| 法說會簡報 | mopsov.twse.com.tw | 最近 5 場 |
| ESG 永續報告書 | esggenplus.twse.com.tw | 最近 3 年 |
| 提升企業價值計劃 | mopsov.twse.com.tw | 依年度 |

## 操作步驟

### Step 1：編輯 Excel

打開 `公司清單.xlsx`，格式如下：

**第 1 列（儲存設定）：**

| A | B |
|---|---|
| 儲存路徑（選填） | *(留空 = 存在本機，填路徑 = 存到指定位置，例如 `G:\我的雲端硬碟\MOPS報告`)* |

**第 2 列（標題列）：**

| A | B | C | D | E | F | G | H | I | J |
|---|---|---|---|---|---|---|---|---|---|
| 股票代碼 | 起始年份 | 結束年份 | 全選 | 年報 | 財報 | 關係企業三書表 | 法說會簡報 | ESG永續報告書 | 提升企業價值計劃 |

**第 3 列起（公司資料）：**

| 股票代碼 | 起始年份 | 結束年份 | 全選 | 年報 | 財報 | 關係企業三書表 | 法說會簡報 | ESG永續報告書 | 提升企業價值計劃 |
|---------|---------|---------|------|------|------|-------------|----------|-------------|--------------|
| 2330 | | | ✓ | | | | | | |
| 2317 | 110 | 115 | | ✓ | | | ✓ | | |
| 3008 | | | | | | | | | ✓ |

- **年份**（B、C 欄）：民國年，選填。都空白預設抓最近 5 年
- **全選**（D 欄）：打 ✓ 下載全部六種報告
- **個別選擇**（E~J 欄）：只在 D 欄留空時有效，打 ✓ 選擇要下載的類型
- **防呆**：D~J 全部留空，自動當作全選

### Step 2：執行下載

雙擊 `批次下載.bat`，等待下載完成。

### Step 3：上傳 GitHub（選用）

雙擊 `push_github.bat`，自動將下載結果 push 到 GitHub。

> 前提：這台電腦需要安裝 Git 並設定好 GitHub 認證（只需設定一次）。

### Step 4：同步 Google Drive（選用）

1. 安裝 [Google Drive 桌面版](https://www.google.com/drive/download/)，登入 Google 帳號
2. 在 `公司清單.xlsx` 的 B1 填入 Google Drive 路徑，例如 `G:\我的雲端硬碟\MOPS報告`
3. 執行批次下載，檔案會直接存到 Google Drive 並自動同步

## 自動跳過已下載的公司

系統會自動檢查每家公司的資料夾是否已有 PDF。有的話就跳過，不會重複下載。

如需重新下載某家公司，**刪除該公司的資料夾**後重跑即可。

## 專案檔案說明

```
mops_pdf_packager/
├── 公司清單.xlsx           ← 你要編輯的 Excel（輸入公司 + 設定路徑）
├── 批次下載.bat             ← 雙擊執行批次下載
├── push_github.bat         ← 雙擊推送到 GitHub
├── push_github.py          ← GitHub 推送邏輯
├── batch_download.py       ← 批次下載主程式
├── main.py                 ← CLI 模式（python main.py 2330）
├── build_exe.bat           ← 打包成 exe 用
├── requirements.txt        ← Python 依賴套件
├── .gitignore              ← Git 忽略規則
└── src/
    ├── scrapers/
    │   ├── ebook_scraper.py              ← 年報/財報/三書表
    │   ├── briefing_scraper.py           ← 法說會簡報
    │   ├── esg_scraper.py                ← ESG 永續報告書
    │   ├── corporate_value_scraper.py    ← 提升企業價值計劃
    │   └── mopsov_scraper.py             ← Selenium 備援
    └── utils/
        └── downloader.py                 ← 下載流程控制
```

## 注意事項

- 下載速度取決於 MOPS 伺服器，請耐心等待
- 內建自動限速（公司間 45~90 秒、年度間 8~15 秒）和重試機制
- 部分公司可能沒有某些報告類型，系統會自動跳過
- 每次執行只保留本次的 log，舊 log 自動清除
